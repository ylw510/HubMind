# Changelog

All notable changes to HubMind will be documented in this file.

## [0.2.0] - 2024-01-XX

### Added
- **DeepSeek model support** - Added DeepSeek as default LLM provider
  - DeepSeek API integration (OpenAI-compatible)
  - Set as default provider for better cost-effectiveness and Chinese support
  - Excellent code generation capabilities

### Changed
- Default LLM provider changed from OpenAI to DeepSeek
- Updated all documentation to reflect DeepSeek as default

## [0.1.0] - 2024-01-XX

### Added
- Initial release of HubMind
- Core features:
  - Trending repository discovery
  - PR analysis and value scoring
  - Natural language issue creation
  - Smart Q&A agent
  - Repository health monitoring
  - Developer dashboard
  - Automation workflows
- CLI interface with rich formatting
- Interactive chat mode
- LangChain integration with Model I/O unified interface
- GitHub API integration
- Multi-LLM provider support (OpenAI, Anthropic, Google, Azure, Ollama, Groq)

### Features
- Get trending repositories by language and time range
- Analyze pull requests with value scoring
- Create issues from natural language
- Ask questions about repositories
- Monitor repository health metrics
- Watch multiple repositories
- Batch operations and automation
